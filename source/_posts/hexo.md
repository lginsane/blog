1. 新增文章

```新增文章
hexo new [layout] <title>
```

简写

```简写
hexo n [layout] <title>
```

2. 生成静态文件

```生成静态文件
hexo generate
```

简写

```简写
hexo g
```

3. 启动服务器

```启动服务器
hexo server
```

简写

```简写
hexo s
```

4. 部署网站

```部署网站
hexo deploy
```

5. 清除缓存文件

```清除缓存文件
hexo clean
```

6. 合并命令 `&&`

```合并命令
hexo clean && hexo deploy
```
